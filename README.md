## Customization

To use this template for a real shop:

1. Update `WHATSAPP_NUMBER` in `main.js` with your real number
2. Update `SHOP_NAME` in `main.js`
3. Replace product images in `images/`
4. Update product data in the `products` array in `main.js`
5. Update the Google Maps embed URL in `index.html` and `contact.html`
